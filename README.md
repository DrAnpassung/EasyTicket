# EasyTicket
A minecraft paper plugin, which adds a ticket system for moderation.


<img width="256" height="256" alt="EasyTicket" src="https://github.com/user-attachments/assets/270acd5a-fbdb-4cde-a0a9-4323b3bebb53" />

## How to use
type '/ticket' to get to the dashboard
if you have the easyticket.staff permission you will be able to see every ticket, otherwise only the ones you made yourself.

you can create tickets by either clicking on the 'create ticket' button in the dashboard and writing the subject in chat or by using this command: /ticket <subject>

its pretty self explanatory im too lazy to write all that :P

## Features
- Tickets in chatroom style
- muted chat while in the ticket
- sorting tickets by 'answered' status or creation/update time
- easy tp to the ticket creator
- ticket log in files; both active and and closed tickets
- configurable maximum amount of tickets per user
- notify staff on ticket updates as well as customizable sounds in the config.yml
